
# 💰 Modabber | Smart Financial Dashboard

**Modabber** is a professional financial dashboard designed with precision to provide a seamless and modern user experience for money management, financial tracking, and data analytics. Built using core technologies (HTML/CSS), this project focuses on intricate details and interactive animations.

---

## ✨ Key Features

- **🎨 Modern UI/UX:** Clean user interface leveraging **CSS Grid** for consistent layout organization.
- **📱 Fully Responsive:** Optimized for all screen sizes, from ultra-wide monitors to mobile devices, using advanced **Media Queries**.
- **💳 Interactive Payment System:** Visa cards featuring a realistic **3D Flip Animation** on hover to reveal CVV and barcode details.
- **📊 Custom Analytics:** Hand-crafted charts built with **CSS Flexbox** for lightweight performance and zero external dependencies.
- **🧩 Smart Components:**
    - **Transactions Table:** Features smart horizontal scrolling on mobile.
    - **Adaptive Sidebar:** Collapses to icons on medium screens and hides on mobile to maximize workspace.
    - **Statistics Cards:** Distinctive color-coded cards for Income, Expenses, and Current Balance.

---

## 🛠 Tech Stack

* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3 (Advanced):**
    * **Grid & Flexbox:** Core layout engines.
    * **Custom Variables:** Centralized management for colors and spacing.
    * **3D Transforms:** Powering the realistic card flip effect.
    * **Responsive Design:** Precise breakpoints for a mobile-first feel.
* **FontAwesome:** For intuitive iconography.
* **Google Fonts:** Utilizing **IBM Plex Sans Arabic** for clear, professional typography.
